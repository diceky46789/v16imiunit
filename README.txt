語順スロット英作文 v16.0

おすすめ方針で、内部構造を整理した安定版です。

主な変更:
・JavaScriptを複数ファイルに分割
・CSVセットの順番管理を getOrderedCsvSets(folderId) に一本化
・フォルダタブ、練習タブ、音声再生タブが同じ順番を使う
・フォルダ内CSVセットを▲▼で並び替え
・並び順はフォルダごとに保存
・CSV内の問題順は維持
・新しいCSVは追加順の最後に追加
・フォルダ名変更、フォルダ移動に対応
・音声再生タブの左右フリックに対応

GitHub Pagesでは、このZIP内のファイルをすべて上書きアップロードしてください。
更新後に反映されない場合は、Safariで再読み込み、または一度ホーム画面アイコンを削除して追加し直してください。
